## GastoViagem-app-android

### 📄 Descrição:

Aplicativo Android, desenvolvido com Kotlin, para efetuar o cálculo de gastos com gasolina em viagens, baseando-se em:

- total de quilometros
- preço por litro
- KM's por litro
